🩸 Acute Lymphoblastic Leukemia Classifier

Welcome to the Leukemia Classifier project! This initiative employs machine learning, computer vision, and various tools to classify Acute Lymphoblastic Leukemia (ALL) images into distinct categories.

🔬 Project Overview

This project utilizes Flask, TensorFlow, and scikit-image to offer a user-friendly interface for leukemia image classification. It integrates two machine learning models for distinguishing between 'hem' and 'all' leukemia types and classifying leukemia stages: 'Benign', 'Early', 'Pre', and 'Pro'.

🌟 Features

  • Image Classification: Upload leukemia images and classify them into specific types and stages.
  • Model Integration: Combines machine learning models to provide accurate classification results.
  • User Authentication: Secure login for authorized access to diagnostic features.

🚀 Setup Instructions

To set up and run this project locally, follow these steps:

git clone https://github.com/nicolasvargaszz/acute-lymphoblastic-leukemia-classifier.git
cd leukemia-classifier
pip install -r requirements.txt
python app.py
Access the app via http://localhost:5000

🧩 Technologies Used

Flask, TensorFlow, scikit-image, SQLite

🔍 Intelligent Inventory Management with Machine Learning

🛠️ Project Structure: CRISP-DM

The project follows the CRISP-DM methodology, consisting of six key steps: Market Understanding, Data Understanding, Data Preparation, Modeling, Evaluation, and Deployment.

📈 Summary

The project uses various methodologies for optimal inventory management. Initially, predictive models and regression techniques were employed, followed by the implementation of Facebook Prophet, which provided significantly improved results.

💻 Machine Learning Models for Complaint Classification

This repository focuses on applying machine learning models to classify complaint types using natural language processing (NLP) techniques. The goal is to predict whether a complaint is related to computer issues or non-computer issues using various ML models.

📂 What Kind of Data Do We Have?

We are working with two important `.csv` files containing computer and non-computer complaints. All complaints are in Spanish, adding complexity to text processing and classification.

🎯 Goal of the Project

Build a machine learning model to accurately classify complaints into computer-related and non-computer-related categories.

🔎 What's the TF-IDF Algorithm?

TF-IDF (Term Frequency-Inverse Document Frequency) is a popular NLP algorithm used to evaluate the importance of a word in a document relative to a collection of documents. This helps in feature extraction and improving model performance.

🧠 Models and Techniques Used

Explore various machine learning models including Naive Bayes, SVM, and Logistic Regression. The project also includes an exploration of neural networks for advanced classification tasks.

📈 Results and Evaluation

Evaluate model performance based on metrics such as accuracy, precision, recall, and F1-score. Compare different models to select the best-performing classifier for the complaint classification task.

🛒 Supermarket Food Basket Comparison
